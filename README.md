## 📌 Sobre os Gráficos

| Card | O que mostra |
|------|-------------|
| **GitHub Stats** | Total de commits, repositórios, seguidores, stars recebidas |
| **Top Langs (compact)** | Porcentagem de código por linguagem em todos os seus repositórios |
| **Top Langs (donut)** | Versão em gráfico de pizza mostrando a proporção de cada linguagem |

> ⚠️ **Importante:** O gráfico mostra a **quantidade de código por linguagem** em bytes, não a quantidade de commits. Para ver commits por linguagem, o GitHub Stats já contabiliza no card principal.
